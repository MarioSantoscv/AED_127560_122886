# Projetos - Algoritmos e Estruturas de Dados

Projetos desenvolvidos no âmbito da unidade curricular de **Algoritmos e Estruturas de Dados**, na **Universidade de Aveiro**, durante o **1.º semestre de 2025/2026**.

## Descrição

Este repositório reúne dois trabalhos práticos focados na implementação, análise e teste de **tipos abstratos de dados (TADs)** e respetivos algoritmos em linguagem C.

### Projeto 1 — TAD `imageRGB`
Desenvolvimento de um TAD para representação e manipulação de imagens em formato RGB, incluindo operações básicas sobre imagens, rotação, comparação e algoritmos de **preenchimento de regiões** e **segmentação**.

### Projeto 2 — TAD `GRAPH`
Desenvolvimento e extensão de um TAD para grafos, com implementação de operações sobre vértices e subgrafos, bem como algoritmos para determinação de **conjuntos de vértices dominantes**, incluindo variantes por cardinalidade mínima e peso mínimo.

## Conteúdos principais

- Implementação de funções incompletas fornecidas nos enunciados
- Testes funcionais e validação dos algoritmos
- Análise de complexidade computacional
- Cuidados com legibilidade, robustez e gestão de memória
